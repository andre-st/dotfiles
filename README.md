# Configuration files available to every system user

- GPL
- don't add sensitive info here, use `~/.XXX_local` files instead
- I'm using the [Notion](https://github.com/raboof/notion) tiling window manager
	- with customized theme
	- with customized keyboard bindings (US-layout), which are mainly centered around 
		the left or right <kbd>Win</kbd>-Key and the area around the cursor keys to 
		switch screens (<kbd>Win</kbd>-<kbd>L</kbd><kbd>R</kbd>), 
		workspaces (<kbd>Win</kbd>-<kbd>U</kbd><kbd>D</kbd>) or 
		tabs (<kbd>Win</kbd>-<kbd>|</kbd))
	- extended by a lua-script for notification sounds (X11 urgency hints) for whitelisted apps such as instant messengers like [Signal](https://github.com/signalapp/Signal-Desktop)
- my approach to long paths or command-line space is a multiline prompt, see [.exports](.exports)
